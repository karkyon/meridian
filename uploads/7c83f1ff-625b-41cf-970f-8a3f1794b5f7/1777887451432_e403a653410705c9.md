### 3.2 Geocoding API仕様

#### エンドポイント
```
GET https://nominatim.openstreetmap.org/search
```

#### リクエストパラメータ
| パラメータ | 型 | 必須 | 説明 |
|-----------|---|------|------|
| q | string | ○ | 検索クエリ（地名） |
| format | string | ○ | レスポンス形式（json） |
| limit | integer | - | 結果件数（デフォルト: 1） |
| accept-language | string | - | 言語（ja） |

#### リクエスト例
```javascript
const query = 'Tokyo Tower';
const url = `https://nominatim.openstreetmap.org/search?q=${encodeURIComponent(query)}&format=json&limit=1`;

fetch(url)
  .then(response => response.json())
  .then(data => {
    if (data.length > 0) {
      const lat = parseFloat(data[0].lat);
      const lon = parseFloat(data[0].lon);
      console.log(`Latitude: ${lat}, Longitude: ${lon}`);
    }
  });
```

#### レスポンス例

```javascript
class Route {
  constructor() {
    this.id = generateRouteId();
    this.positions = [];
    this.totalDistance = 0;
    this.startTime = new Date();
    this.endTime = null;
  }

  addPosition(position) {
    if (this.positions.length > 0) {
      const lastPos = this.positions[this.positions.length - 1];
      this.totalDistance += position.distanceTo(lastPos);
    }
    this.positions.push(position);
  }

  end() {
    this.endTime = new Date();
  }

  getDuration() {
    if (!this.endTime) return null;
    return (this.endTime - this.startTime) / 1000; // 秒
  }
}
```

### 6.2 Nominatim連携

#### レート制限対応
```javascript
class GeocodingService {
  constructor() {
    this.lastRequestTime = 0;
    this.MIN_INTERVAL = 1000; // 1秒
  }

  async search(query) {
    const now = Date.now();
    const elapsed = now - this.lastRequestTime;
    
    if (elapsed < this.MIN_INTERVAL) {
      await this.sleep(this.MIN_INTERVAL - elapsed);
    }
    
    this.lastRequestTime = Date.now();
    
    // API呼び出し
    return fetch(`https://nominatim.openstreetmap.org/search?q=${query}&format=json`);
  }

  sleep(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

## 7. エラー処理

### 7.1 エラー分類

| エラーコード | エラー名 | 説明 | 対応 |
|------------|---------|------|------|
| ERR-001 | WebGL未対応 | ブラウザがWebGLに対応していない | エラーメッセージ表示 |
| ERR-002 | Cesium初期化失敗 | CesiumJSの初期化に失敗 | リトライ or エラー表示 |
| ERR-003 | Geocoding失敗 | 地名検索でエラー | エラーメッセージ表示 |
| ERR-004 | 地名未検出 | 検索結果が0件 | 「見つかりません」表示 |
| ERR-005 | 地形データ取得失敗 | Cesium Ionからのデータ取得失敗 | リトライ or 代替データ |
| ERR-006 | LocalStorage失敗 | データ保存・読込失敗 | エラーログ出力 |

### 7.2 エラーハンドリング実装

```javascript
class ErrorHandler {
  static handle(errorCode, error) {
    console.error(`[${errorCode}]`, error);
    
    const errorMessages = {
      'ERR-001': 'お使いのブラウザはWebGLに対応していません。Chrome、Edge、Safariの最新版をご利用ください。',
      'ERR-002': 'アプリの初期化に失敗しました。ページをリロードしてください。',
      'ERR-003': '地名検索中にエラーが発生しました。',
      'ERR-004': '指定された地名が見つかりませんでした。別の地名で検索してください。',
      'ERR-005': '地形データの取得に失敗しました。通信環境を確認してください。',
      'ERR-006': 'データの保存に失敗しました。'
    };
    
    this.showErrorMessage(errorMessages[errorCode] || 'エラーが発生しました。');
  }
  
  static showErrorMessage(message) {
    // UIにエラーメッセージを表示
    const errorDiv = document.getElementById('error-message');
    errorDiv.textContent = message;
    errorDiv.style.display = 'block';
    
    setTimeout(() => {
      errorDiv.style.display = 'none';
    }, 5000);
  }
}
```

```javascript
class InputValidator {

    // HTMLタグを除去
    const temp = document.createElement('div');
    temp.textContent = query;
    const sanitized = temp.innerHTML;
    
    // スクリプトタグを除去
    return sanitized.replace(/<script[^>]*>.*?<\/script>/gi, '');
  }
  
  static validateCoordinates(lat, lon) {
    if (typeof lat !== 'number' || typeof lon !== 'number') {
      return false;
    }
    
    if (lat < -90 || lat > 90) return false;
    if (lon < -180 || lon > 180) return false;
    
    return true;
  }
}
```
